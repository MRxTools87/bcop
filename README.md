# bcop

## Installation

JUST COPY PASTE YOUR TERMUX.


```bash
wget https://raw.githubusercontent.com/MRxTools87/bcop/refs/heads/main/bcop && chmod u+x bcop && mv bcop $PREFIX/bin/
```

## Usage


─────────────────■Usage■──────────────────

#     [**bcop**  [-hv] [-b|-r [home|pkgs]]

────────────────────────────────────────

1. -h              print version
2. -v            print this usage
3. -b      backup home and/or packages
4. -r      restore home and/or packages
 
────────────────────────────────────────
